# A simple Router Project with Django4

##### First Create a env with pipenv and install PipFile package

you can Start the project with this commond :
```python manage.py runserver```

These Path are Created :
| Page | Link |
| ---- | ---- |
| Home | / |
| AboutUs | /about |
| 404Page | /Any Url |
